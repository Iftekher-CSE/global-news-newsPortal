Question 01: What are the difference between var, let and const?

Answer: The scope of a var variable is functional scope. The scope of a let variable is block scope. The scope of a const variable is block scope.
Var can be updated and re-declared. Let can be updated but cannot be re-declared. const cannot be updated or re-declared.
Var and let can be declared without initialization but const cannot be declared without initialization.

Question 02: What are the difference between arrow function and function expression?

Answer: Regular JavaScript function has it's own dynamic value of this. But the arrow function doesn't have it's own this.
Arguments objects are not available in arrow functions, but are available in regular functions.
If the return statement is missing, or there's no expression after return statement, the regular function implicitely returns undefined. If the arrow function contains one expression, and you omit the function's curly braces, then the expression is implicitly returned.

Question 03: What are the benefits of using template string?

Answer: Template strings are a powerful feature of modern JavaScript released in ES6. It lets us insert/interpolate variables and expressions into strings without needing to concatenate. It allows us to create strings that are complex and contain dynamic elements.
Template Literals use back-ticks (``).
With template literals, you can use both single and double quotes inside a string.
Template literals allows multiline strings.
Template literals allow variables in strings.
Template literals provide an easy way to interpolate variables and expressions into strings.
